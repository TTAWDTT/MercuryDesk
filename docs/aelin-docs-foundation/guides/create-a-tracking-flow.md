# Create a Tracking Flow

## 示例流程

1. 你提问：`最近 yetone 在 X 上有什么新动态？`
2. Aelin 回答并给出证据
3. Aelin 询问是否需要长期跟踪
4. 你确认后，Aelin 自动检查账号配置并启动同步
5. 后续变化会进入跟踪中心和通知中心

## 建议实践

- 跟踪目标要尽量具体（账号 / 网址 / 唯一标识）
- 为不同主题使用不同 workspace
- 定期清理失效跟踪，保持记忆质量
