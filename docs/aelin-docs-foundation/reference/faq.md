# FAQ

## Q1: Aelin 为什么有时会联网、有时不会？
A: 默认是模型自动决策。问题需要最新事实时更可能联网；纯历史/本地上下文问题更偏向本地检索。

## Q2: 跟踪是不是自动开启？
A: 不会。Aelin 会建议，最终由你确认。

## Q3: 为什么有时只看到检索结果没有深度回答？
A: 一般与模型配置异常或证据不足有关。先检查 LLM 连通性，再重试。

## Q4: 可以自定义 LLM 提供商吗？
A: 可以，支持自定义 Provider/URL/Model。

## Q5: 记忆会不会无限增长？
A: 当前会按分层策略和摘要机制控制规模，后续会继续优化压缩与淘汰规则。
