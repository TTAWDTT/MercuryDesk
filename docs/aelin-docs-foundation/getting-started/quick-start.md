# Quick Start

## 1. 启动服务

- 后端启动后，访问前端页面。
- 登录账号，确保 `Agent Config` 中已配置可用模型（例如 DeepSeek）。

## 2. 发送第一条问题

示例：

```text
今天勇士和马刺的比赛结果如何？
```

Aelin 会自动决定是否使用：

- 本地检索
- 网络检索
- 两者并行

## 3. 确认跟踪

当 Aelin 判断你可能需要长期关注时，会提示是否跟踪。

- 选择“跟踪”：进入长期跟踪流程。
- 选择“仅这次”：只回答当前问题，不建立持续任务。

## 4. 查看结果沉淀

你可以在以下位置看到沉淀：

- 对话证据卡片
- 跟踪中心
- 记忆层（facts / preferences / in_progress）

## 5. 第二次追问

示例：

```text
那库里最近 5 场状态有没有波动？
```

Aelin 会结合上次结果与新检索继续回答，而不是重新从零开始。
