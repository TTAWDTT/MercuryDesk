# Welcome to Aelin

Aelin 的核心不是“一次性搜索”，而是“长期跟踪 + 持久记忆 + 可对话推理”。

## Aelin 解决什么问题

- 你关注的信息分散在多个平台，来回切换很累。
- 普通 Chat 只能临时搜索，下一次对话又要从头来。
- 你需要一个能持续记住你关注对象、并主动提醒变化的助手。

## Aelin 与普通 Chat 的区别

- 普通 Chat：按次搜索，短期上下文。
- Aelin：支持本地历史 + 网络搜索 + 跟踪订阅，形成长期记忆。

## 三个关键词

- `Answer-first`：先回答问题，再给出后续动作建议。
- `Long-term memory`：把高价值信息持续写入记忆层。
- `Tracking`：你确认后，Aelin 自动持续跟踪目标对象。
