# Web Search + Local Memory

## 双检索模式

- `Local`: 查询你已同步和已存储的数据
- `Web`: 查询外部公开网页并抽取正文
- `Auto`: 模型自动决策是否并行触发

## 为什么这样做

- 只用本地：可能缺少最新信息
- 只用网络：缺少你的历史偏好与上下文
- 双检索融合：更完整，也更贴近你的长期关注

## 证据输出

- 每条关键结论尽量关联证据卡片
- 支持直接打开来源网页（预览或外链）
