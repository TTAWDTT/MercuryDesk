# Long-term Tracking

## 跟踪流程

1. Aelin 提出跟踪建议
2. 用户确认
3. 检查数据源配置
4. 启动同步任务
5. 持续写入消息与记忆

## 当前支持方向

- Web 跟踪（公开信息）
- 平台型跟踪（如 X / 抖音 / 小红书 / 微博 / B 站，取决于你的配置）
- 邮件与 RSS 类源

## 可见性

- 跟踪中心可查看条目状态
- 记忆层中可看到进行中事项
- 通知中心会展示状态变化
